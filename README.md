Overview

This repository showcases cybersecurity projects, lab environments, and technical exercises that demonstrate practical, hands-on security skills. Each project is designed to demonstrate not only technical implementation, but also the ability to document processes, explain design decisions, troubleshoot complex issues, and communicate results effectively.

Rather than serving as a collection of notes, this repository functions as a professional portfolio of completed work. Many of these projects are accompanied by articles published on the Hupfen Dynamics blog that explore the design decisions, challenges, and lessons learned behind each build.
