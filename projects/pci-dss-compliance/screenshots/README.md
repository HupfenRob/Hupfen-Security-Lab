# PCI DSS-Inspired Security Operations Lab Screenshots

This directory contains screenshots documenting the implementation and validation of the **Building and Validating a PCI DSS-Inspired Security Operations Lab** project.

The screenshots provide visual evidence of key configuration milestones, telemetry validation, intrusion detection, and security monitoring throughout the environment. Together with the architectural diagrams and implementation notes, they demonstrate that each major security control was successfully deployed and verified.

| Screenshot | Description |

| `vmware_network_adapters.png` | Multi-network adapter configuration for the pfSense virtual machine supporting segmented lab networks. |
| `pfsense_dhcp_configuration.png` | DHCP configuration for one of the segmented pfSense interfaces. |
| `splunk_enterprise_dashboard.png` | Splunk Enterprise successfully deployed as the centralized SIEM. |
| `splunk_windows_event_validation.png` | Windows endpoint telemetry successfully indexed within Splunk. |
| `suricata_eve_json_output.png` | Structured EVE JSON telemetry generated by Suricata. |

## Purpose

These screenshots document the progression of the project from infrastructure deployment through telemetry collection, intrusion detection, and security validation. They provide supporting evidence that the environment was successfully configured and that each major security control functioned as intended during controlled testing.
