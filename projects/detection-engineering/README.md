# Detection Engineering Program

## Overview

This project demonstrates the process of building, validating, and maintaining security detections using Splunk, Windows Event Logs, Sysmon, and Suricata. Throughout the project, detections are developed, tested, tuned, and documented using a repeatable workflow that reflects real-world detection engineering practices.

## Objectives

- Understand attacker behavior
- Identify appropriate telemetry
- Build Splunk detections
- Validate detections
- Tune detections
- Create alerts
- Document detection logic
