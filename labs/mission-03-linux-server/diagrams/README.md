# Mission 3 Diagrams

This directory contains architectural diagrams created for Mission 3 of the Hupfen Security Lab.

The diagrams illustrate the overall design of the Linux security server, highlighting the major services configured during this mission and how they contribute to a secure foundation for future lab activities.

## Contents

| Diagram | Description |
|----------|-------------|
| `linux_security_server_overview.png` | High-level overview of the Ubuntu security server, including OpenSSH, auditd, journald, and AIDE, and how these services establish the foundation for future centralized logging and security operations. |

## Purpose

These diagrams provide architectural context for the implementation steps documented in the mission README and complement the supporting screenshots by illustrating how the server components work together.
