# Mission 3 Scripts

This directory contains scripts developed during Mission 3 to automate recurring Linux administration and security tasks.

The scripts support baseline creation, integrity monitoring, and future validation activities performed throughout the Hupfen Security Lab.

## Contents

| Script | Purpose |
|--------|---------|
| `capture_baseline_hashes.sh` | Captures SHA-256 hashes of critical Linux system files to establish a trusted integrity baseline for future comparison. |
