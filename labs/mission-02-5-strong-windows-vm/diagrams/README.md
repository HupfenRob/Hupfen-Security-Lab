# Mission 2.5 Diagrams

This directory contains architectural diagrams that illustrate the overall design and security architecture of the hardened Windows workstation created during Mission 2.5.

These diagrams provide a high-level overview of the workstation's layered security controls, endpoint protections, and telemetry sources, complementing the detailed implementation documented in the mission README.

| `secure_windows_workstation_overview.png` | High-level overview of the hardened Windows workstation, including administrative separation, endpoint protection, security telemetry, and its role in supporting future security operations within the Hupfen Security Lab. |
