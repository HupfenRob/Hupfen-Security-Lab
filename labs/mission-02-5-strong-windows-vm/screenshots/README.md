# Mission 2.5 Screenshots

This directory contains screenshots documenting the major implementation steps completed during Mission 2.5.

The images are referenced throughout the mission README to illustrate the workstation hardening process, security control configuration, validation activities, and endpoint monitoring capabilities.

## Contents

| Screenshot | Purpose |
|------------|---------|
| `creating_initial_user.png` | Creating the initial local user account for the workstation. |
| `itadmin_account.png` | Configuring a dedicated administrative account to support the principle of least privilege. |
| `powershell_password_rotation.png` | Demonstrating automated local administrator password rotation using PowerShell. |
| `windows_event_viewer.png` | Verifying Windows Event Viewer and security event logging. |
| `group_policy_editor.png` | Configuring Local Group Policy to apply workstation hardening and security settings. |
